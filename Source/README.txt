This is the source folder. 
If you are running on Windows, we recommend you use the compiled version.
---------------------------------------------------------

If you wish to run this program from the source:

-Install Python 3
-Install PyQt4 for your version of python.

-Do one of the following:
	*Double-click on gui.py
	*open a python interpreter and run gui.py
	*In a terminal, navigate to this folder and type 
		"python gui.py" or "py gui.py" or a similar command. 

---------------------------------------------------------

It is also possible to compile the file if you have cx_Freeze installed. 
If cx_Freeze is installed and properly configured on your system, 
then build setup.py to create a folder with a compiled version of the code.
